# 🌲 Timber Game

A simple and interactive **2D Timber Game** built using **C++ and SFML**, where the player chops wood while avoiding falling branches. The game tests reflexes, timing, and quick decision-making skills.

---

## 🎮 Gameplay

* Chop wood from the left or right side
* Avoid falling branches 🌿
* Score increases with each successful chop
* Game ends when:

  * Player gets hit ❌
  * Time runs out ⏱️

---

## 🕹️ Controls

* **← Left Arrow** → Move Left
* **→ Right Arrow** → Move Right
* **Enter** → Start / Restart Game

---

## ⚙️ Tech Stack

* **Language:** C++
* **Library:** SFML (Simple and Fast Multimedia Library)

## 🚀 How to Run

1. Install **SFML** on your system
2. Configure SFML in your IDE (Visual Studio / CodeBlocks)
3. Compile and run `Timber1.cpp`
